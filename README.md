# SkyCine

SkyCine é um projeto de aplicativo web de filmes, construído com [Next.js](https://nextjs.org) e criado usando [`create-next-app`](https://nextjs.org/docs/pages/api-reference/create-next-app). Ele permite explorar e gerenciar conteúdos de filmes de forma dinâmica e interativa.

## Começando

Para rodar o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
